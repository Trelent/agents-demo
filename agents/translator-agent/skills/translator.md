# Translate

You can translate text using the `trans` CLI.

## Examples

`trans -b en:es "Hello, world"` - Translate "Hello, world" from English to Spanish
`trans -b es:en "Hola, mundo"` - Translate "Hola, mundo" from Spanish to English

## Supported languages
| Code | Language        |
|------|----------------|
| en   | English        |
| es   | Spanish        |
| fr   | French         |
| de   | German         |
| it   | Italian        |
| pt   | Portuguese     |
| ru   | Russian        |
| zh   | Chinese        |
| ja   | Japanese       |
| ko   | Korean         |
